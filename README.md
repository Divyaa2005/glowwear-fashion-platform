# 🌟 GlowWear – Women’s Fashion Platform

GlowWear is a modern women’s fashion web platform designed to help users discover stylish outfits at affordable prices. The platform curates products from popular e-commerce sites like Myntra and Flipkart and presents them in a clean, premium, and user-friendly interface.

## 🚀 Features
* 🛍️ Browse fashion categories:
  * Dresses
  * Shoes
  * Makeup
  * Bags
  * Jewellery

* 💖 Wishlist functionality
  Save your favorite items for later.

* 🔐 User Authentication
  Users can sign up and log in securely.

* 🔗 External Product Links
  Each product includes a “Buy Now” button that redirects users to platforms like Myntra or Flipkart.

* 📱 Fully Responsive Design
  Works smoothly across mobile, tablet, and desktop.

* 🎨 Premium UI
  Inspired by modern, high-end fashion websites.

## 🧠 How It Works

Users can explore different categories and view curated fashion items with prices and details.
When a user clicks on "Buy Now", they are redirected to the original shopping website to complete the purchase.

The platform focuses on **affordable fashion discovery**, making it easier for users to find trendy outfits within their budget.

## 🛠️ Tech Stack
